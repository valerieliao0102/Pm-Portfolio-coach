# PM Portfolio Coach 🎯

An AI-powered tool that helps Product Managers build a complete portfolio from their resume — not just summarize it.

## Live Demo

👉 [https://valerieliao0102.github.io/pm-portfolio-coach](https://valerieliao0102.github.io/pm-portfolio-coach)

## What It Does

Most PM portfolios are just resumes in disguise. This tool is different.

1. **Upload your resume (PDF)** → AI analyzes your background
2. **AI breaks down your projects** → Identifies 2–3 case studies from your experience, and suggests what's missing from your portfolio
3. **Conversational Q&A** → AI asks you questions one by one to help you tell each project story fully (Problem → Action → Results)
4. **Generate portfolio PPT** → Downloads a ready-to-use `.pptx` file

## How to Use

1. Open [the tool](https://valerieliao0102.github.io/pm-portfolio-coach)
2. Upload your resume as a PDF
3. Enter your [Anthropic API Key](https://console.anthropic.com) *(used only in your browser, never stored)*
4. Pick a color theme
5. Click **Start Analysis**
6. Go through the Q&A for each project
7. Download your portfolio PPT


## Requirements

- An Anthropic API Key ([get one here](https://console.anthropic.com))
- A PDF resume with text content (scanned images won't work)

## Local Development

No setup needed. Just open `index.html` in your browser.
